# scala-utils.nvim

Some extra utils for working with modern Scala tooling in NeoVim.

NOTE: This is pretty rapidly changing at the moment, so expect things to break.

## Installation

```lua
use {
  "ckipp01/scala-utils.nvim",
  requires = { "nvim-lua/plenary.nvim" }
}
```
